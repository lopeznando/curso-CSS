Objetivo

el objetivo de crear mi archivo estilos.css es para separar la estructura del diseño

  • asi tendremos mi archivo .html solo para la estrctura elementos etiquetas y contenido
  • mi archivo .css solo para los selectores y propiedades de diseño que le asignare a cada etiqueta

Modo de uso

para relacionar un archivo html con su archivo css (hojas de estilo). debemos hacer uso de nuestra etiqueta link en la cabezera del documento. y en donde se encuentre el archivo.
